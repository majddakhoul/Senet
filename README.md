# Senet

A classic Senet board game project with AI, allowing players to move pieces along the S-path. Includes special houses, strategic rules, and an AI opponent with evaluation algorithms to challenge the player.

## Features
- Two-player mode (human vs AI)
- AI opponent with configurable search depth
- Special houses affecting gameplay (Happiness, Rebirth, Horus, Water, etc.)
- Board display and turn management
- Legal move validation
